Getting Started

Docker (Recommended):

cd awesome-langchain-langgraph
docker compose up -d

Then open http://localhost:8080 in your browser.

The Lifecycle Journey

Week 1: 🔨 BUILD — Pick LangChain (models + tools) → Choose LangGraph (orchestration) → Use a template → Wire integrations

Week 2: 🔭 OBSERVE — Enable LangSmith tracing → Open LangGraph Studio → Watch every tool call

Week 3: 📊 EVALUATE — Build a test dataset → Run openevals scoring → Gate release on score

Week 4: 🚀 DEPLOY — Choose LangGraph Platform (or Docker) → Add Agent Chat UI → Launch production

Finding What You Need

Example 1: Research Agent

  1. Click 🔨 build
  2. Look for: Core Ecosystem (LangChain + LangGraph), Templates (Retrieval Agent), Community — Research Agents
  3. Click 🔭 observe to add LangSmith + Studio
  4. Ready to ship? Click 🚀 deploy

Example 2: Evaluate Your Agent

  1. Click 📊 evaluate
  2. See: LangSmith (evaluation platform), openevals (ready-made scorers), agentevals (agent-specific scoring)
